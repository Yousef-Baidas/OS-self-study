Annotated code blocks use `astro-expressive-code` — a filename tab, a
highlighted line, and diff markers, here with `fork()` creating a child process:

```c title="fork.c" {5} ins={7}
#include <unistd.h>

int main() {
  // Fork a new process:
  pid_t pid = fork();

  if (pid == 0) {
    printf("child\n");
  }

  return 0;
}
```

<Callout type="key">
  `fork()` returns **twice** — `0` in the freshly created child, and the child's
  PID in the parent. That single return value is how each process learns which
  one it is.
</Callout>
